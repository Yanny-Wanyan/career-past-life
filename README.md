# Career Past Life

> *Answer 10 immersive ancient-China scenario questions and discover your past-life official title*

## About

A fun, single-page personality quiz styled as an ancient Chinese imperial appointment. Users answer 10 scenario-based questions set in a fictional dynasty, and receive a personalized result: an official title, a full "past life biography", personality stats, and workplace CP (best partner & nemesis).

**Live demo:** https://yanny-wanyan.github.io/career-past-life/

## Features

- 10 immersive story-driven questions with instant witty commentary
- 15 result types (including extreme results "Emperor" & "Happy Commoner" + hidden "Warlord" easter egg)
- Full past-life biography timeline
- Workplace CP matching (best ally & worst enemy)
- Animated stat bars
- Ancient decree-style result card with image export (long-press on mobile / download on desktop)

## Tech

Pure frontend, single HTML file:

- HTML / CSS / Vanilla JS
- [html2canvas](https://html2canvas.hertzen.com/) for result card export
- Google Fonts (Ma Shan Zheng & Noto Serif SC)
- Hosted on GitHub Pages
